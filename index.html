Installation locale

TikZJax compile du vrai LaTeX/TikZ en WebAssembly dans le navigateur. La lib doit être servie localement — les assets WebAssembly ne peuvent pas venir d'un CDN tiers.

Mise en place en 3 étapes

  1. Décompresser le ZIP fourni : tous les fichiers (tikzjax.js, run-tex.js, core.dump.gz, tex.wasm.gz, fonts/, tex_files/) doivent être dans le même dossier que votre HTML.
  2. Ajouter dans le <head> :
    <link rel="stylesheet" href="fonts.css">
    <script src="tikzjax.js"></script>
  3. Ouvrir la page via Live Server (VS Code) ou tout serveur HTTP local — le protocole file:// bloque les requêtes WebAssembly (CORS).

Tableaux de variations

Le package tkz-tab génère des tableaux de signes et de variations avec une syntaxe concise. Les symboles d, z, t encodent respectivement discontinuité, zéro et valeur intermédiaire.

tkz-tab Polynôme du second degré — \( f(x) = x^2 - 4 \)
Lecture : \(f\) décroît jusqu'au minimum \(-4\) en \(x=0\), puis croît. z indique un zéro de \(f'\).
Code LaTeX
% Activer tkz-tab avec data-tex-packages
\begin{tikzpicture}
  \tkzTabInit[lgt=3, espcl=2.5]
    {$x$          / 1,
     $f'(x) = 2x$ / 1,
     $f(x)$       / 2}
    {$-\infty$, $0$, $+\infty$}
  \tkzTabLine{, -, z, +, }
  \tkzTabVar
    {+/$+\infty$,
     -/$-4$,
     +/$+\infty$}
\end{tikzpicture}
tkz-tab Fonction rationnelle — \( f(x) = \dfrac{x^2-1}{x} \) avec valeurs intermédiaires
Code LaTeX
\begin{tikzpicture}
  \tkzTabInit[lgt=3.5, espcl=2.8]
    {$x$     / 1,
     $f'(x)$ / 1,
     $f(x)$  / 2.5}
    {$-\infty$,$-1$,$0$,$1$,$+\infty$}
  \tkzTabLine{, -, z, +, d, -, z, +, }
  \tkzTabVar
    {+/$0^-$,
     -/$-2$,
     +D-/$+\infty$/$-\infty$,
     +/$2$,
     -/$0^+$}
  %% Valeurs intermédiaires
  \tkzTabVal[draw]{1}{2}{0.5}
    {$-\sqrt{2}$}{$-\sqrt{2}$}
  \tkzTabVal[draw]{4}{5}{0.5}
    {$\sqrt{2}$}{$\sqrt{2}$}
\end{tikzpicture}

Tableaux de signes

Les tableaux de signes combinent plusieurs lignes de facteurs avec le signe du produit. Indispensable pour résoudre inéquations et étudier le signe d'une expression.

tkz-tab Signe de \( f(x) = \dfrac{(x-1)(x+3)}{x-2} \)
Code LaTeX
\begin{tikzpicture}
  \tkzTabInit[lgt=3, espcl=2]
    {$x$    / 1,
     $x+3$ / 1,
     $x-1$ / 1,
     $x-2$ / 1,
     $f(x)$ / 1}
    {$-\infty$,$-3$,$1$,$2$,$+\infty$}
  \tkzTabLine{, -, z, +, , +, , +, }
  \tkzTabLine{, -, , -, z, +, , +, }
  \tkzTabLine{, -, , -, , -, d, +, }
  \tkzTabLine{, -, z, +, z, -, d, +, }
\end{tikzpicture}

Courbes & fonctions

TikZ trace des courbes paramétriques, implicites, polaires et cartésiennes avec une précision vectorielle. Les bibliothèques calc et intersections permettent des constructions géométriques exactes.

TikZ Courbe, tangente et aire sous la courbe
Code LaTeX
\begin{tikzpicture}[scale=1.1]
  %% Axes
  \draw[->] (-0.3,0)--(3.8,0) node[right]{$x$};
  \draw[->] (0,-0.3)--(0,3.5) node[above]{$y$};
  %% Aire sous la courbe (remplissage)
  \fill[blue!15] (1,0)
    -- plot[domain=1:3,samples=60](\x,{0.3*\x*\x})
    -- (3,0) -- cycle;
  %% Courbe f(x) = 0.3x²
  \draw[blue,very thick]
    plot[domain=0.1:3.5,samples=80](\x,{0.3*\x*\x});
  %% Tangente en x=2 : y=1.2(x-2)+1.2
  \draw[red,thick]
    plot[domain=0.5:3.5](\x,{0.6*2*(\x-2)+0.3*4});
  \fill[red] (2,{0.3*4}) circle (2pt);
\end{tikzpicture}
TikZ Courbe paramétrique (cycloïde) & courbe polaire (lemniscate)
Astuce clé
%% Courbe paramétrique x(t), y(t)
\draw[teal,very thick]
  plot[domain=0:2*pi, samples=120, smooth]
  ({(\x - sin(\x r))},
   {(1 - cos(\x r))});

%% Les angles en radians : sin(\x r)
%% sign() donne le signe du cos
\draw[purple,very thick]
  plot[domain=0:2*pi, samples=200]
  ({2*cos(\x r)*sqrt(abs(cos(2*\x r)))
    *sign(cos(2*\x r))},
   {2*sin(\x r)*sqrt(abs(cos(2*\x r)))
    *sign(cos(2*\x r))});
Astuce Utilisez samples= pour contrôler la finesse. Pour les courbes avec des points anguleux, augmentez jusqu'à 300–400.

Géométrie plane

Les bibliothèques angles, patterns et decorations.pathreplacing permettent de produire des figures de géométrie au niveau Lycée avec marquages d'angles, hachures et accolades cotées.

TikZ Triangle, angles, hauteur, hachures et accolade
Code LaTeX
\usetikzlibrary{angles,quotes,patterns,
  decorations.pathreplacing}
\begin{tikzpicture}[scale=1.15]
  \coordinate (A) at (0,0);
  \coordinate (B) at (4,0);
  \coordinate (C) at (1.2,3);
  \coordinate (H) at (1.2,0);
  %% Hachures
  \fill[pattern=north west lines,
        pattern color=blue!30] (A)--(B)--(C)--cycle;
  %% Angle avec arc
  \pic[draw, angle radius=12pt,
       "$\alpha$" font=\small]{angle=B--A--C};
  %% Accolade cotée
  \draw[decorate,
    decoration={brace,mirror,amplitude=6pt}]
    (A)--(H) node[midway,below=8pt]{$d_1$};
  %% Flèche double pour mesure
  \draw[|-|,red,thick] (C)--(H)
    node[midway,right=3pt]{$h$};
\end{tikzpicture}
TikZ Cercle trigonométrique avec valeurs remarquables
Technique : boucle foreach
%% Placer automatiquement les angles
\foreach \a/\lbl/\pos in {
  30/{$\frac{\pi}{6}$}/above right,
  45/{$\frac{\pi}{4}$}/above right,
  60/{$\frac{\pi}{3}$}/above right,
  ...
}{
  \fill[gray!60]
    ({cos(\a)},{sin(\a)}) circle (1.2pt);
  \node[\pos,font=\tiny]
    at ({1.18*cos(\a)},{1.18*sin(\a)}) {\lbl};
}
%% Projections du point M(π/3)
\draw[red,dashed]
  ({cos(60)},0)--({cos(60)},{sin(60)});
\draw[teal,dashed]
  (0,{sin(60)})--({cos(60)},{sin(60)});

PGFPlots — Graphes statistiques

PGFPlots gère histogrammes, nuages de points, diagrammes en barres, et courbes sur axes gradués avec légendes. Idéal pour les stats en Terminale.

pgfplots Histogramme & courbe de densité
Code LaTeX

  \begin{axis}[
    width=7cm, height=5.5cm,

    xmin=0, xmax=21,
    ymajorgrids=true,
    grid style={dashed,gray!30},
    bar width=0.9,

    \addplot[ybar, fill=teal!50] coordinates {
      (2.5,2)(7.5,5)(12.5,8)
      (17.5,7)(19.5,3)(21,5)
    };
  \end{axis}
\end{tikzpicture}
pgfplots Nuage de points & droite de régression
Code LaTeX
\begin{axis}[
  xmin=0, xmax=11,
  ymin=0, ymax=22,
  grid=major,
  legend pos=north west
]

  \addplot[only marks, mark=*,

      (1,3)(2,4.5)(3,7)(4,8)(5,10)

\end{axis}

Graphes & arbres

La bibliothèque graphs trace des graphes de la théorie des graphes, trees génère des arbres de probabilités. Essentiel pour les chapitres combinatoires.

TikZ · trees Arbre de probabilités — tirage successif sans remise
Code LaTeX
\begin{tikzpicture}[
  level 1/.style={
    sibling distance=3.5cm,
    level distance=2cm},
  level 2/.style={
    sibling distance=1.8cm,
    level distance=2cm},
  edge from parent/.style={draw,thick}
]
\node[circle,draw,fill=gray!15]{$\Omega$}
  child {
    node[circle,draw,fill=blue!20] {$R_1$}
    child {
      node[rect,draw] {$R_1\cap R_2$}
      edge from parent
        node[left]{$\frac{2}{5}$}
    }
    child { ... }
    edge from parent
      node[left]{$\frac{3}{6}$}
  }
  child { ... };
\end{tikzpicture}
Astuce Les styles level 1/.style et level 2/.style contrôlent l'espacement à chaque profondeur. Ajustez sibling distance selon le nombre de branches.
TikZ · graphs Graphe pondéré — algorithme de Dijkstra
Code LaTeX
\usetikzlibrary{positioning}
\begin{tikzpicture}[
  node/.style={circle,draw,thick,
    minimum size=28pt},
  marked/.style={circle,draw,thick,
    fill=teal!30,minimum size=28pt}
]
  \node[marked] (A) at (0,0)   {$A$};
  \node[node]   (B) at (2.5,1.2){$B$};
  \node[node]   (C) at (2.5,-1.2){$C$};
  ...
  %% Arête avec poids
  \draw[thick] (A)--(B)
    node[midway,above]{4};
  %% Chemin optimal en rouge
  \draw[red,very thick] (A)--(C)
    node[midway,below,red]{2};
\end{tikzpicture}

Probabilités & loi normale

PGFPlots trace la courbe de Gauss et colorie les zones de rejet/confiance indispensables pour les tests statistiques et intervalles de fluctuation.

pgfplots Loi normale \( \mathcal{N}(0,1) \) — zone de rejet à 5%
Code LaTeX
\begin{axis}[
  xmin=-3.8, xmax=3.8,
  axis x line=bottom,
  axis y line=none,
  xtick={-1.96,0,1.96}
]
  %% Zone centrale (95%)
  \addplot[blue!60,fill=blue!12,
    domain=-1.96:1.96,samples=100]
    {exp(-x^2/2)/sqrt(2*pi)} \closedcycle;
  %% Zones de rejet (2×2.5%)
  \addplot[red!80,fill=red!20,
    domain=-3.8:-1.96,samples=60]
    {exp(-x^2/2)/sqrt(2*pi)} \closedcycle;
  \addplot[red!80,fill=red!20,
    domain=1.96:3.8,samples=60]
    {exp(-x^2/2)/sqrt(2*pi)} \closedcycle;
  %% Courbe de densité
  \addplot[blue,very thick,
    domain=-3.8:3.8,samples=150]
    {exp(-x^2/2)/sqrt(2*pi)};
\end{axis}
Formule La densité de \(\mathcal{N}(0,1)\) : exp(-x^2/2)/sqrt(2*pi).
Pour \(\mathcal{N}(\mu,\sigma)\) : exp(-(x-mu)^2/(2*s^2))/(s*sqrt(2*pi)).

Figures en 3D

La bibliothèque tikz-3dplot définit des repères 3D avec transformation de vue. PGFPlots peut aussi tracer des surfaces.

tikz-3dplot Repère 3D — vecteurs, plan et sphère
Code LaTeX
%% Angle de vue : élévation 65°, azimut 125°
\tdplotsetmaincoords{65}{125}
\begin{tikzpicture}[tdplot_main_coords,
               scale=1.3]
  %% Axes en 3D
  \draw[thick,->] (0,0,0)--(2.5,0,0)
    node[anchor=north east]{$x$};
  \draw[thick,->] (0,0,0)--(0,2.5,0)
    node[anchor=north west]{$y$};
  \draw[thick,->] (0,0,0)--(0,0,2.5)
    node[anchor=south]{$z$};
  %% Plan (x,y)
  \fill[blue!10,opacity=0.5]
    (0,0,0)--(2,0,0)--(2,2,0)--(0,2,0)--cycle;
  %% Vecteur
  \draw[red,very thick,->]
    (0,0,0)--(1.2,1,1.5) node[right]{$\vec{v}$};
  %% Projeté (pointillés)
  \draw[dashed,gray] (1.2,1,0)--(1.2,1,1.5);
\end{tikzpicture}

Formules MathJax avancées

MathJax 3 avec les packages AMS couvre tout le niveau CPGE : matrices, systèmes, intégrales multiples, sommes, produits, suites.

MathJax Formulaire de référence — niveau Terminale / CPGE
Dérivées & intégrales
\[\int_a^b f(x)\,\mathrm{d}x = \Big[F(x)\Big]_a^b\] \[\frac{\mathrm{d}}{\mathrm{d}x}\left(\frac{u}{v}\right) = \frac{u'v - uv'}{v^2}\]
Matrices & déterminants
\[\det\begin{pmatrix}a&b\\c&d\end{pmatrix} = ad-bc\] \[A^{-1} = \frac{1}{\det A}\,\mathrm{com}(A)^{\top}\]
Probabilités & dénombrement
\[\binom{n}{k} = \frac{n!}{k!\,(n-k)!}\] \[P(A|B) = \frac{P(A\cap B)}{P(B)}\] \[\sum_{k=0}^{n}\binom{n}{k}p^k(1-p)^{n-k}=1\]
Suites & séries
\[\sum_{k=1}^{n}k = \frac{n(n+1)}{2}\] \[\lim_{n\to+\infty}\left(1+\frac{x}{n}\right)^n = e^x\]
Intégration dans vos pages : MathJax et TikZJax cohabitent parfaitement dans la même page. Utilisez \( … \) pour les formules inline et \[ … \] pour les formules centrées. Les deux systèmes se chargent indépendamment — aucun conflit.
TikZ · automata Automate fini déterministe — reconnaissance de \( a^*b^+ \)
Code LaTeX
\usetikzlibrary{automata,positioning}
\begin{tikzpicture}[auto,>=stealth,
  every state/.style={draw,circle,
    minimum size=28pt}]
  \node[state,initial,initial text={}]
    (q0) {$q_0$};
  \node[state,right=of q0]
    (q1) {$q_1$};
  \node[state,accepting,right=of q1]
    (q2) {$q_2$};    % état final : double cercle
  %% Transitions
  \path[->] (q0) edge node {$a$} (q1)
                 edge[loop above]
                   node {$a$} (q0);
  \path[->] (q1) edge node {$b$} (q2);
  \path[->] (q2) edge[loop above]
                   node {$b$} (q2);
\end{tikzpicture}
