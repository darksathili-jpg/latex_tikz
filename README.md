# latex_tikz
Il s'agit d'une page de documentation interactive et didactique, conçue à destination des enseignants de mathématiques (lycée, CPGE). Elle illustre comment intégrer des figures LaTeX/TikZ directement dans un navigateur web, sans serveur ni compilation, grâce à TikZJax (exécution en WebAssembly) et MathJax 3 pour les formules.
