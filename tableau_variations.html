1. Rappels sur \( f(x) = \dfrac{1}{x} \)

La fonction est définie sur \( \mathbb{R}^* = ]-\infty\,;\,0[ \cup ]0\,;\,+\infty[ \).

Sa dérivée est \( f'(x) = -\dfrac{1}{x^2} < 0 \) pour tout \( x \in \mathbb{R}^* \) : la fonction est strictement décroissante sur chacun de ses intervalles de définition.

Attention La décroissance est séparée sur \(]-\infty\,;\,0[\) et \(]0\,;\,+\infty[\). On ne peut pas « traverser » la discontinuité en \(0\).

3. Tableau de variations de \( f(x) = \dfrac{1}{x} \)
Lecture
  • Sur \(]-\infty\,;\,0[\) : \(f\) décroît de \(0^-\) vers \(-\infty\).
  • En \(x=0\) : double barre de discontinuité.
  • Sur \(]0\,;\,+\infty[\) : \(f\) décroît de \(+\infty\) vers \(0^+\).

5. Comment utiliser dans tes fiches
Structure des fichiers requise ta_fiche.html doit être dans le même dossier que les fichiers du package : tikzjax.js, fonts.css, run-tex.js, core.dump.gz, tex.wasm.gz, fonts/, tex_files/.
Syntaxe dans le HTML Dans le <head> : charger fonts.css et tikzjax.js localement.
Dans le <body> : écrire le code tkz-tab dans <script type="text/tikz" data-tex-packages='{"tkz-tab": ""}'>.
Important Servir via Live Server ou tout serveur HTTP — les chemins relatifs ne fonctionnent pas en file:// (restriction CORS du navigateur). Avec Live Server de VS Code, c'est transparent.
